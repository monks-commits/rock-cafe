# Днепропетровское Рок Кафе 2001-2021

Архивный веб-сайт легендарного рок-кафе Днепропетровска.

## Описание

Этот сайт является архивным проектом, созданным для сохранения памяти о Днепропетровском Рок Кафе, которое существовало с 2001 по 2021 год. Сайт содержит историю всех локаций кафе, фотографии, воспоминания и материалы, связанные с этим легендарным местом.

## Структура проекта

### HTML файлы:
- `index.html` - Главная страница с обзором всех локаций
- `about.html` - Подробная информация о проекте и концепции кафе
- `contacts.html` - Контактная информация и информация о сборе материалов
- `pobeda.html` - Локация "Победа" (2001-2011)
- `lenina.html` - Локация "Ленина" (2011-2013)
- `leningradskaya.html` - Локация "Ленинградская" (2013-2018)
- `oldschool.html` - Old School Café (2018-2019)
- `geroev.html` - Локация "Героев Сталинграда" (2019-2021)

### CSS и JavaScript:
- `styles.css` - Основные стили с темной ностальгической темой
- `script.js` - Функциональность переключения языка и вкладок

## Особенности

### Дизайн
- Темная ностальгическая тема
- Цветовая схема: черный, серый, бордовый, золотой
- Адаптивный дизайн для всех устройств
- Анимации и переходы

### Функциональность
- Двуязычный интерфейс (русский/английский)
- Переключатель языка в шапке
- Система вкладок на страницах локаций
- Чат-заглушка для публики
- Плавная прокрутка и анимации

### Навигация
- Главное меню с переходами между страницами
- Карточки локаций на главной странице
- Вкладки на страницах локаций:
  - История
  - Интерьер
  - Концерты-афиши
  - Гости

## Использование

1. Откройте `index.html` в браузере
2. Используйте переключатель языка в шапке для смены языка
3. Кликайте по карточкам локаций для перехода на соответствующие страницы
4. На страницах локаций используйте вкладки для просмотра разного контента
5. Чат-заглушка доступна на всех страницах

## Технические требования
